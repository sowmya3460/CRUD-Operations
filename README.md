# 🎓 Student Performance Prediction System

A Flask-based web application that performs CRUD operations without a database and uses Machine Learning to predict student performance (Pass/Fail).

## 🚀 Features
- Add, Update, Delete, View student records
- ML model predicts Pass/Fail
- Data stored in CSV file (no database)
- Simple dashboard UI
