# 🎓 Student Performance Prediction System

A Flask-based web application that performs CRUD operations without a database and uses Machine Learning to predict student performance (Pass/Fail).

## 🚀 Features
- Add, Update, Delete, View student records
- ML model predicts Pass/Fail based on study hours, attendance, and assignments
- Data stored in CSV file (no database)
- Simple and clean dashboard UI

## 🛠️ Tech Stack
- Flask
- Python (Pandas, NumPy, Scikit-learn)
- HTML, CSS, Bootstrap
- CSV file for storage

## ▶️ Run Project
```bash
pip install flask pandas numpy scikit-learn joblib
python train_model.py
python app.py
